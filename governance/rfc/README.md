# نظام إدارة التغيير للدراسات - RFC System

## كيفية إنشاء RFC جديد

1. انسخ قالب RFC: `cp RFC-template.md RFC-YYYYMMDD-<slug>.md`
2. املأ جميع الحقول المطلوبة
3. احصل على موافقة اللجنة المختصة
4. حدث `registry.json` عند الموافقة
5. نفذ التغييرات حسب الجدول الزمني

## حالات RFC مطلوبة
- أي تعديل على دراسة موجودة
- إضافة دراسة جديدة
- تغيير في معايير الجودة
- تحديث في متطلبات الأمان
- تغييرات في النموذج التجاري
